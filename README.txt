# Свадебный лендинг (Ольга & Артём)

## Что внутри
- index.html — сайт
- bg.jpg — фон (ваше фото)
- story-1.jpg / story-2.jpg — детские фото

## Как запустить
Откройте index.html в браузере.

## Как выложить на GitHub Pages
1) Создайте репозиторий на GitHub
2) Загрузите туда файлы из этого архива
3) Settings → Pages → Deploy from branch → main / root
